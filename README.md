# 🎮 坦克大战 (Tank Battle)

经典 FC 红白机 **Battle City** 风格的坦克大战游戏，纯 HTML5 Canvas 实现。

## 🎯 在线游玩

**[▶️ 开始游戏](https://lllhhhrrr317.github.io/tank-battle/)**

## 🕹️ 操作方式

| 操作 | 按键 |
|------|------|
| 移动 | `↑↓←→` 或 `WASD` |
| 射击 | `空格` |
| 暂停 | `P` |

📱 支持移动端触摸操作

## 🎮 游戏特色

- 🎨 经典 Battle City 风格画面
- 🤖 多种敌人 AI，难度递进
- 🧱 可破坏砖墙 + 不可破坏钢铁墙
- 🌊 水域障碍 / 🌲 树林掩护
- 🦅 保卫基地（老鹰）机制
- 💎 5 种道具：⭐加分、🛡加命、⏱时停、💣清屏、🛡护盾
- 🔊 Web Audio API 音效
- 📊 分数与生命系统
- 📱 移动端触摸操作
- 💥 粒子特效与屏幕震动

## 🚀 本地运行

直接浏览器打开 `index.html` 即可。

## 📂 技术栈

- HTML5 Canvas
